🦄 🦄 Unicorn Company Analysis (SQL Data Analytics Project)
📌 Project Overview

This project performs an in-depth SQL analysis of global Unicorn Companies (startups valued at $1B+) to uncover business insights related to growth, funding efficiency, industry dominance, and geographic expansion.

The objective is to simulate real-world business analysis using structured SQL queries and analytical thinking.

🎯 Business Objectives

This project answers key strategic questions:

Which countries produce the highest number of unicorns?

Which industries dominate the unicorn ecosystem?

How long does it take for startups to reach unicorn status?

Which companies are most capital-efficient?

Which year experienced peak unicorn growth?

How does funding relate to valuation?
